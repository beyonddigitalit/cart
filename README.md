## Installation

You can install the package via composer:

```bash
composer require beyonddigitalit/cart
```

## Usage

``` php
$skeleton = new Beyonddigitalit\Cart();
echo $skeleton->echoPhrase('Hello, Beyonddigitalit!');
```

## Testing
